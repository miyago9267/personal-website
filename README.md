# Miyago Personal Site

A modern personal landing page built with Vite + Vue 3 + UnoCSS, optimized for Bun.

**Live Site**: <https://info.miyago9267.com>

## Getting started

```bash
bun install
bun run dev
```

## Scripts

- `bun run dev` - local dev server
- `bun run build` - production build
- `bun run preview` - preview build

## Structure

- `src/sections/` - page sections (hero, about, experience, contact)
- `src/components/ui/` - reusable UI atoms
- `src/data/` - content data and profile info
- `src/legacy/` - previous components kept for reference
